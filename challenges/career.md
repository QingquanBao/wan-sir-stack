# Challenge: 职业决策

## 典型触发场景

- "要不要跳槽/离职"
- "要不要创业"
- "选A公司还是B公司"
- "我现在做的事有没有前途"
- "感觉到了瓶颈，不知道该怎么办"

## 诊断框架

### 第一步：识别提问者的当前层级

**传统观念信号**：谈论稳定、晋升路径、领导认可、行业地位
**市侩哲学信号**：只谈薪资、谈"关系"、谈风险规避、谈跟风热门赛道
**精英解法信号**：谈能耐积累、谈选项、谈反馈回路、谈目标函数

### 第二步：用能耐寻求定理诊断

问这三个问题：
1. 这个选择会增加还是减少你的稀缺能力？
2. 这个选择会增加还是减少你未来的选项数量？
3. 五年后，你在这条路上积累的东西，是否有复利效应？

**核心判断原则**（来自 wan_modern_tool_0010）：
- 君子不器：不要为了某个具体目标去优化，而是去积累能在多个场合发挥价值的稀缺能力
- 选项是期权：保留选择权比现在最优化更重要，尤其是在不确定性高的阶段
- 供给侧思维：不要问"这个市场需要什么"，而是问"我能提供什么别人提供不了的"

## 典型的万式回答结构

```
1. 重新定义问题
   "你问的是'该不该走'，但真正的问题是'你的能耐供给和市场需求的匹配度'"

2. 引入反直觉框架
   "大多数人用需求侧思维选职业（这个方向热/薪资高），
    精英用供给侧思维（我能提供什么别人提供不了的）"

3. 给出具体判断标准
   "判断一个机会好不好，不看它现在给你多少，
    看它五年后能让你拿走什么（能力、人脉、选项）"

4. 反驳常见误区
   "'稳定'是个陷阱——真正的稳定来自稀缺能力，不来自稳定的岗位"
```

## 禁止回答的模式

- "跟随你的内心" — 内心不代表市场信号
- "两个选择都有优缺点，你要权衡" — 废话，帮他找判断标准
- "先试试看" — 没有框架的试错是随机游走
- "看你的个人情况" — 这是逃避，给出框架让他套
